input의 종류
종류 모양 설명 주요속성
text 한줄짜리 텍스트
  • size:화면에 보여줄 크기
  • value:텍스트박스에 들어갈 값
  • maxlength:입력할 수 있는 최대 문자 수
password 입력내용 마스킹
search 검색어 입력
url url 유효성 검사
email email 유효성 검사
tel 키보드 어댑테이션(모바일기기)
checkbox 복수선택
  • value:서버에 넘겨줄 값(필수)
  • checked:기본선택
radio 단일선택
number 숫자만 입력 가능, 스핀박스로 숫자 조절 가능
  • min:최대값
  • max:최소값
  • step:증감단위
range 숫자 조절을 위한 슬라이드 막대
date 연,월,일 선택
month 연,월 선택
week 연,월,주 선택
time 시,분 선택
datetime-local 연,월,일,시,분 선택
file 파일 첨부
hidden 사용자에게는 보이지 않지만 서버로 넘겨주는 값 입력
submit form을 서버로 전송
reset form을 리셋
image 버튼 대신 사용할 이미지
button 일반 버튼
