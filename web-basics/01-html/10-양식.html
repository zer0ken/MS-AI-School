요소 설명 주요속성
여러 줄 입력 : textarea
  • cols:가로 단위를 문자 단위로 지정
  • rows:세로 길이를 줄 단위로 지정
  • maxlength:최대 글자 수
드롭다운 목록 : select, option select의 속성
  • size:화면에 보여줄 드롭다운 항목의 개수
  • multiple:둘 이상의 항목 선택
option의 속성
  • selected:선택
데이터 목록 : textlist, option input의 list속성과 datafield의 id 속성을 동일하게 지정
button : css를 이용하여 꾸밀 수 있다.
  • style : submit/reset/button
